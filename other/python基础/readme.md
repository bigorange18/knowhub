### 3. python内存管理

1. 私有堆空间： 所有python的对象都存储在这个堆中。
2. 引用计数：当引用被创建时候加一；当引用被删除时，计数减一；当某个计数降为0时候，会被立即回收。
3. 垃圾回收：当多个对象相互引用时候形式环，导致技术永远不能为0。**标记-清除：**阶段1（标记）：从一组“根对象”（如当前运行的函数、全局变量等）出发，遍历所有可以访问到的对象，并标记为“存活”。阶段2（清除）： 遍历整个堆，将所有未被标记的对象视为“垃圾”，并回收它们的内存。这解决了循环引用问题。
